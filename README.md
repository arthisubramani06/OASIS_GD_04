Task 4: Infographic Design

Objective: Create a visually informative infographic to present data or concepts clearly
Tools Used: Canva / Illustrator / Photoshop
Skills Applied: Data visualization, layout design, color coding, information hierarchy

Description:
The infographic was designed to present complex information in a simple, engaging, and easy-to-understand format. Emphasis was on clarity, readability, and visual appeal.

Learning Outcomes:
Learned how to simplify complex information using graphics
Improved understanding of visual hierarchy and color psychology
Gained experience in combining data and design for effective communication
